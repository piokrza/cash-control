# CashControl

Technologies used:
- Angular 17,
- NgRx,
- Firebase,
- PrimeNg,
- Typescript,
- PrimeFlex,
- Hardhat,
- Ethers.js

Work in progress...

App built on Node v20.6.1
run `npm install --legacy-peer-deps` for installing dependencies,
run `ng s` for a dev server.
run `npx hardhat node` to run local blockchain
